# rshell

A simple shell implementation written in Rust, created as a learning exercise.

## Building and Running

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd rshell
    ```
2.  **Build the project:**
    ```bash
    cargo build
    ```
3.  **Run the shell:**
    ```bash
    cargo run
    ```

## Supported Commands


*   `exit`
*   `ls`
*   `echo`
*   `pwd`
*   `cd`
*   `touch`
*   `rm`
*   `cat`
